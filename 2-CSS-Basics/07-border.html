Borders

The border property allows you to specify the style, width, and color of an elements border.

Border Style

The border-style property can take up to four valies (top border, right border, bottom border, left border). The following border-style values are allowed:

  • dotted:dotted border
  • dashed:dashed border
  • solid:solid border
  • double:double border
  • groove:groove border
  • ridge:ridge border
  • inset:inset border
  • none:no border
  • hidden:hidden border
  • mixed:you can set a different border style for top, right, bottom, and left

Border Width

VERY IMPORTANT: border-width will not work unless you have a border-style set!!

Border-width can be set to a specific size (px, em,etc) or you can use one of the pre-set settings: thin, medium, thick. The border-width property can also take up to four values (top,right, bottom,left)
