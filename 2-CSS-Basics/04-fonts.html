Font

In CSS, there are two types of font fammily names:

  • generic family a group of font families with a similar look (ex: Serif)
  • font family a specific font family (ex: "Times New Roman", or Arial)

The font-family property should hold several font names as you fallback in case the browser doesn't support the first font you picked.
Always start with the font you eant, and end witha generic family. Also, if the font family is more than one word, it must be in
quotationmarks (ex: "Times New Roman").

Here is a list of the CSS font properties:

font:
sets all the font propeties in one declaration
font-family:
specifies the font family for text
font-size:
specifies the font size of a text
font-style
specifies the font style of a text
font-weight:
specifies the weight of a font
