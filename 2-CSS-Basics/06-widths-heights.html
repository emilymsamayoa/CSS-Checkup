Width and Height Notes

The width and height of properties are used to set the widthnd height of an element.

The default setting for width and height is auto, which means the browser decided the width and height

Other setting for width and height are:

  • specified length alues such as px & em
  • percent of the containing div

This element has a width of 50% and a height of 200px.

You can also manipulate width an height with images. This image was originally approx 400px x 200px, now it is closer to 400px x 400px.
