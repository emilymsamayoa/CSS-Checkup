Position

The position propery specifies the type of positioning method used for an element. The four possition vlaues are:

  • fixed:this will pin something to a spot, an dit will stay there as you scroll. (ex: nav bar)
  • static:this is the defou;f setting, it is not affected by top, bottom,, left, and right properties. It is always positioned according to the normal flow of the page.
  • relative:does not take the element out of normal document flow. Still it appears within its space/div/container, but allows it to be offset a bit from the top, bottom, right, left. Top, right, botttom, and left properties will move it away from its normal position. Other content will not be adjusted to fit into any gap left by the element.
  • absolute:Takes an element out of the normal document flow. If a parent element position is set to relative, and the child element is set to absolute, the child element's ne (0,0) coordinate will be the upper left hand cornor of the parent element instead of the whole page.

The Z-index property allows you to override the normal stacking order of the page. The default z-index is 0, and you can set that value to be positive or negative. an element witha greater z-index will stack in front of an element wit a lower z-index. If two elements have the same z-index or dont have one assigned, the element positioned last in the HTML will show on top. You must have a position assigned in order to assign a z-index.
